# Contexto para agentes de código

Este repositorio tiene documentación de referencia en [`openwiki/`](./openwiki/README.md). Antes de explorar todo el código fuente desde cero, revisá ahí:

- `openwiki/architecture.md` — stack, modelo de datos, componentes.
- `openwiki/android-packaging.md` — cómo está envuelto con Capacitor y cómo generar el `.apk`.
- `openwiki/decisions-log.md` — fixes y decisiones ya tomadas (para no repetir diagnóstico).

Si modificás `src/` de forma significativa, actualizá `openwiki/architecture.md` para que siga reflejando la realidad del código.
